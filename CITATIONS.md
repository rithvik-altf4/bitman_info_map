# Citations

This project collects course information for the BITMAN program (Common Year 1 + Analytics/Data Management option) from BCIT’s public website.

## Program pages (used to identify courses / outlines)
- BCIT — Business Information Technology Management (BITMAN) Diploma (Full-time): https://www.bcit.ca/programs/business-information-technology-management-diploma-full-time-6235dipma/
- BCIT — BITMAN: Analytics & Data Management option (Full-time): https://www.bcit.ca/programs/business-information-technology-management-analytics-data-management-option-diploma-full-time-623cdipma/

## Course outline pages (used for outline links)
- BCIT Course Outlines directory (individual course outline pages, e.g. `/outlines/babi4005/`): https://www.bcit.ca/outlines/


## Notes on how the data was pulled
- Course codes were taken from the program pages and/or course outline links found on those pages.
- Course names and prerequisite text were pulled from the corresponding BCIT course page.
- Output data was saved into the project CSV generated by the notebook.
